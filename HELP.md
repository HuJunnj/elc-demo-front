# 💻 E-Commerce Frontend (React + Taro + TypeScript)

这是一个使用 React + Taro + TypeScript 构建的跨端电商前端项目，支持 Web 和小程序。

---

## 📦 技术栈

- React
- Taro 4.x
- TypeScript
- Axios（或 fetch）

---

## 🚀 启动方式

### 前提要求

- Node.js 22+
- npm 或 pnpm 10.9.2
